# Task Organization

## Overview
This feature defines how tasks are organized, searched, and filtered.

## Fields
Task includes:
- priority (low, medium, high)
- tags
- due_date

## Capabilities

### Set Priority
The user must be able to assign a priority level to a task.

### Tag Tasks
The user must be able to assign one or more tags (e.g., Work, Home, Study).

### Search Tasks
The user must be able to search tasks by title or description.

### Filter Tasks
The system must allow filtering tasks by:
- completed status
- priority
- tag
- due date

### Sort Tasks
The system must allow sorting tasks by:
- due date
- priority
- alphabetical order
