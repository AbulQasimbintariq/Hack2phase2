# Task CRUD

## Overview
This feature defines the core task management capabilities of the system.

## Entities
- Task
- User

## Fields
Task includes:
- id
- user_id
- title
- description
- completed
- created_at
- updated_at

## Capabilities

### Add Task
The system must allow a user to create a new task with a title and optional description.

### Update Task
The system must allow a user to modify the title and description of an existing task.

### Delete Task
The system must allow a user to permanently remove a task.

### View Task List
The system must return all tasks belonging to a user.

### Mark as Complete
The system must allow toggling the completed status of a task.
